Pengaturan Sinkronisasi Google Sheets

Gunakan panel ini untuk mengaktifkan sinkronisasi otomatis ke Google Sheets. Anda hanya perlu menyalin tautan Web App Apps Script milik Anda di sini.

Langkah Membuat Integrasi Google Sheets:

  1. Buka Google Sheet Baru.
  2. Di baris pertama (Header), siapkan kolom-kolom berikut: No Invoice | Tanggal | Nama Klien | No Telp | Rincian Item | Total Tagihan | Uang Muka (DP) | Sisa Pelunasan
  3. Klik menu **Ekstensi** > **Apps Script**.
  4. Hapus semua kode bawaan lalu tempel kode (script) di bawah ini:
    5. function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    
    sheet.appendRow([
      data.invoiceNo,
      data.date,
      data.clientName,
      data.clientPhone,
      data.itemsSummary,
      data.total,
      data.dp,
      data.balance,
      new Date()
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ "status": "success" }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
  5. Klik tombol **Terapkan (Deploy)** di pojok kanan atas > **Penerapan Baru (New Deployment)**.
  6. Pilih Jenis sebagai **Aplikasi Web (Web App)**.
  7. Ubah akses dari "Hanya saya" menjadi **"Siapa saja" (Anyone)**.
  8. Klik **Terapkan**, lalu setujui otorisasi akun Google Anda.
  9. Salin **URL Aplikasi Web** yang diberikan, lalu paste ke kotak isian di atas!
